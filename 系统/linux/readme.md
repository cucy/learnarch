## 内存布局

- 文本: 程序指令
- 数据: 程序使用的静态变量
- 堆: 程序可从该区域动态分配额外内存
- 栈: 随函数调用, 返回而增减的一片内存, 用于局部变量和函数调用链接信息分配存储空间
